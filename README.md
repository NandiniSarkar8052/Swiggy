# Swiggy Sales Analysis Dashboard

This project analyzes Swiggy order data using Python and creates visual dashboards to highlight revenue trends, sales performance, and item categories.

## What it does
- Loads Swiggy order data from an Excel file
- Cleans and transforms the data using `pandas`
- Calculates key metrics such as total sales, average ratings, and average order value
- Generates visualizations for:
  - Daily revenue trends
  - Quarterly performance
  - Revenue by state
  - Top 5 cities by sales
  - Veg vs Non-Veg revenue share

## Tools used
- Python
- pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## How to run
1. Place the Swiggy dataset file in the workspace, for example: `swiggy_data.xlsx`
2. Open `code.ipynb` in Jupyter or VS Code
3. Run the notebook cells sequentially

## Notes
- The notebook includes data cleaning, feature engineering, grouping, aggregation, and chart generation.
- Update the dataset path in the notebook if your file is located elsewhere.
